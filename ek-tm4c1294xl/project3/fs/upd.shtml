Update

To update the firmware, do the following:

  1. Start the LM Flash Programmer application
  2. Click the "Configuration" tab and set the "Quick Set" choice to "Manual Configuration - see below."
  3. Set the "Interface" choice to "Ethernet Interface."
  4. Enter the parameters given below as the client IP and MAC addresses.
  5. Click the "Program" tab and chose the binary file containing the new firmware image by pressing the "Browse..." button.
  6. Press the "Program" button. If after a few seconds the status bar doesn't show the update in progress, press the "Update" button below.
